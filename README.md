# CV_Builder
Conversational CV Builder

Enable NTT DATA employees to create a CV by interacting with a conversational interface: either a chat bot (text) or a speech-enabled interface (voice). The system will capture the conversation, convert speech to text where needed, take the CV in word, PPT or PDF guide the user to complete all required fields, and produce a NTT DATA CV in the active template version. If no standard template is supplied, the AI should propose template options with NTT DATA logo, allow selection, preview, and then generate the final CV in the selected template.
 
Text chat (web, mobile, messaging platforms) and speech (web, mobile, IVR) flows for CV creation.
Speech-to-text (S2T) transcription, guided prompts, and contextual follow-ups.
Template suggestion engine when no template is given.
Mapping created content to the canonical schema and generating formatted outputs (PDF/DOCX/JSON).
Validation, preview, edit, and human-in-the-loop review.
Multilingual support and accessibility features.
